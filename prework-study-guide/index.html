HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The li can be as many as needed in between the ul.
  • img sets an image on the webpage.
  • src sets the location of the image
  • lang is the language that the webpage is written in.
  • id identifies elements and selects them.
  • br is a line of empty space or line break.
  • a stands for anchor and creates links to same webpage or other webpages.
  • footers appears at the bottom of the page usually containing author, copyright, contact, sitemap and navigation.

CSS

  • Style=color:red followed by header changes the color of the header. Don't forget the ""
  • style followed by h1{color} tells all elements with h1 becomes red
  • link is used when using an external style sheet placed in HTML's head
  • div {color: blue; text-align: justify;}
  • div=Selector color: blue;=Declaration text-align: justify;=Declaration
  • margin acts like margin in word document. Indicates how much space we want around the outside of an element
  • padding gets confused for margin but is the INSIDE of the element. Makes it bigger
  • * declared all of the elements will have style properties margin and padding set as 0
  • link element creates connection to external life
  • rel attribute specifies the relationship between the current document and the linked document
  • href attribute specifies the location(URL) of the external resource

Git

  • git staus: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • add adds modifications in current branch
  • -A adds all changes
  • commit puts a commit message describing what the commit contains
  • pull recieves a branch modifications into local environment
  • push pushes changes we have locally to our remote GitHub branch

Javascript

  • To declare a variable use the var keywork. After var you give it a unique name
  • use an assignment operator = to assign specific data you want to store ina variable
  • var shape = "circle"
  • command option i(Mac) brings up Chrome DevTools-brings up code on right side of screen
  • if statements are if a certain requirment is met and does whatever is in the curly brackets
  • else if comes after the if statement regarding the rest
  • for statements are for loops, elements that will continously change
